# age-detector

ウェブカメラを使用して検出された顔の年齢と性別を推定するシンプルなデモアプリです。

## デモ

https://code4fukui.github.io/age-detector/

## 参考

- ライブラリ: [face-api.js](https://github.com/justadudewhohacks/face-api.js)

## 機能

- ウェブカメラのビデオストリームから顔を検出
- 検出された顔の年齢と性別を推定
- 最も大きく映っている顔の年齢、性別、信頼度を表示

## 要件

なし

## 使い方

1. Webブラウザでデモページを開きます。
2. ウェブカメラへのアクセスを求められたら、許可します。
3. 検出された顔の年齢と性別が画面に表示されます。
