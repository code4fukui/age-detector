# age-detector

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A simple demo app that uses a webcam to estimate the age and gender of detected faces.

## Demo

https://code4fukui.github.io/age-detector/

## Reference

- Lib: [face-api.js](https://github.com/justadudewhohacks/face-api.js)

## Features

- Detects faces in a webcam video stream
- Estimates the age and gender of the detected faces
- Displays the age, gender, and confidence level for the largest face

## Requirements

None

## Usage

1. Open the demo page in a web browser.
2. Grant permission to access the webcam when prompted.
3. The age and gender of the detected faces will be displayed on the screen.
